# BookSpace - Online Bookstore

## Description

An e-commerce bookstore where users can browse, filter, and purchase books across mutiple genres. Administrator can easily manage and update the book inventory.

### Tech Stack
- Frontend: React, Vite
- Backend: Express.js, Node.js
- Styling: CSS3
- State Management: useReducer
- Image Upload: Multer 
- Cloud: Cloudinary
- Database: MySQL (Dockerized)


#### Features
* --User Features--
- User Registration, login, logout,  add items, and checkout functionality
- Browse and filter books by category (Business Strategy, Fiction, Non-fiction, Fantacy)
- Add items to shopping cart with quantity management and delete item functionality
- Update personal information, password, change password, and view oreder history

* --Admin Features--
- Manage products with add new item, edit or delete existing product (CRUD)
- Image upload to Cloudinary
- Edit existing product details (price, description, genre..etc)

##### How to use
* --User--
1. New user register an account with email
2. Browse books by category (Business Strategy, Fiction, Non-fiction, Fantasy)
3. Click "Add to Cart" to add books to shopping cart
4. Cart icon shows item count when closed
5. Click Cart icon to view cart details
6. Click "Checkout" to complete purchase and clear cart
7. Click "My Account" to update personal information, change password, and view order history

* --Admin--
1. Login
2. Add, edit, delete products from inventory
3. Click "Add new item" to create a new product  
4. Save changes or cancel editing
5. Click "Edit" to modify existing products
6. Click "Delete" to remove products
7. Click "My Account" to update personal information, change password
